# 👗 POS Magasin — Point of Sale System

A complete **Point of Sale** web application for a small clothing store.  
Built with pure HTML, CSS and JavaScript — no backend required.

## 🌐 Live Demo
👉 **[Open the app](https://YOUR_USERNAME.github.io/pos-app/)**

## ✨ Features
- 🛒 Barcode/manual product scanning & sales
- 📦 Stock management with category filters
- ↩️ Returns & refunds
- 📊 Daily revenue dashboard
- 👥 CRM — customer management & loyalty points
- 💬 Customer feedback
- 📥 Reports & CSV export (sales, products, customers)
- 💱 WhatsApp & Email communication

## 🚀 Run locally
Just open `index.html` in your browser — no installation needed.  
All data is stored in `localStorage`.

## 📝 Demo data
Sample products, customers and sales are automatically loaded on first launch.
